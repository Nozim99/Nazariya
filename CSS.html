static

Viewport

Fixed

Absolute

sticky

Element scrollning qayerda joylashganiga qarab joylashadi. Bu qiymat relative va fixed qiymatlarini o'zida saqlaydi va scrollni qayerda joylashganiga qarab shu ikki qiymatdan birini o'ziga oladi

Transition

Elementimizni xossa qiymatlarini ko'rsatilgan vaqt favomida ravon o'zgarishini ta'minlaydi. Transition yaratish uchun unga ikki qiymat ko'rsatilishi shart: effekt qo'shmoqchi bo'lgan elementimiz xossasi va effektning davomiyligi. Effektimiz davomiyligi son ko'rinishidagi qiymatga ega bo'lib, o'lchov birligi s (soniya) yoki ms (millisoniya)larda ko'rsatilishi kerak


transition: xossa-1 davomiyligi-1, xossa-2 davomiyligi-2, ..., xossa-n davomiyligi-n


Hamma CSS xossalari ham transition effektiga ega bo'la olmaydi. Ko'rsatilgan havola orqali qaysi xossalarga transition effektini ko'rsatish mumkin ekanligini bilib olishingiz mumkin

Transition timing function

Effektimizni tezligi qanday egri chiziqga ega bo'lishini belgilab beradi

ease - effektimiz sekin boshlanib, keyin tezlashib va yana sekinlik bilan tugaydi. Bu default qiymat
linear - effektimiz ko'rsatilgan vaqt davomida bir xil tezlikga ega bo'ladi
ease-in - effektimiz sekin boshlanadi
ease-out - effektimiz sekin tugaydi
ease-in-out - effektimiz sekin boshlanib va sekin tugaydi
cubic-bezier(x,y,z,k) - cubiz-bezier funksiyasi orqali, ko'rsatilgan egri chiqiz effektiga ega bo'ladi. x, y, z o'rniga qiymat ko'rsatiladi


transition-timing-function: ease;

Transition delay

Effektimiz qachon boshlanishini (kechikishini) belgilab beradi. Son ko'rinishidagi qiymatga ega bo'lib, o'lchov birligi s (soniya) yoki ms(millisoniya)larda ko'rsatilishi kerak


transition: ...; transition-delay: kechikish-davomiyligi;

CSS animation

Elementni bir "style"dan boshqa "style"ga asta - sekinlik bilan o'tishini ta'minlaydi. Animation istalgan CSS xossani istalgan martta o'zgartirish imkonini beradi

@keyframes

Animatsiya stillarni qaysi vaqtda qaysi qiymatga o'zgartirishini belgilab beradi

@keyframes animatsiya-nomi {
   keyframe-intervali-1 {
     xossa-1: qiymat-1;
     xossa-2: qiymat-2;
     . . .
     xossa-n: qiymat-n;
   }

   . . .

  keyframe-interval-n {
    xossa-1: yangi-qiymat-1;
    xossa-2: yangi-qiymat-2;
     . . .
    xossa-n: yangi-qiymat-n;
  }
}


Animatsiyamizga xoxlagan nomimizni bersak bo'ladi. Keyframe interval uchun esa 0-100%ga bo'lgan qiymatlarni ko'rsatib o'tishimiz mumkin. 0 va 100% uchun from va to kalit so'zlaridan foydalanishimiz mumkin

@keyframes box-mover {
  0% {
    left: 0;
    background-color: red;
    width: 200px;
  }

  100% {
    left: 500px;
    background-color: green;
    width: 400px;
  }
}

Animation duration

Animatsiyani qanday davomiylikga ega bo'lishini belgilab beradi. Son ko'rinishidagi qiymatga ega bo'lib, o'lchov birligi s (soniya) yoli ms (millisoniya)larda ko'rsatilishi kerak

Animation delay

Animatsiyamiz qachon boshlanishini (kechikishini) belgilab beradi. Son ko'rinishidagi qiymatga ega bo'lib, o'lchov birligi s (soniya) yoki ms (millisoniya)larda ko'rsatilishi kerak

animation-delay: 2s;

Animation iteration count

Animatsiyamiz necha marta qaytarilishini belgilab beradi. Default qiymati 1 ga teng

animation-iteration-count: 2;

Animation direction

Animatsiya oldinga (forwards), orqaga (backwards) yoki almashinib (alternate) qo'yilishini belgilab beradi

normal - Animatsiya oldinga qarab qo'yiladi

reverse - Animatsiya orqaga qarab qo'yiladi

alternate - Animatsiya birinchi oldinga keyin esa orqaga qarab qo'yiladi

alternate-reverse - Animatsiya birinchi orqaga keyin esa oldinga qarab qo'yiladi


animation-direction: reverse;

Animation timing function

Animatsiyamiz tezligi qanday egri chiziqga ega bo'lishini belgilab beradi

ease - animatsiya sekin boshlanib, keyin tezlashib va yana sekinlik bilan tugaydi (default qiymat)

linear - animatsiya ko'rsatilgan vaqt davomida bir xil tezlikga ega bo'ladi

ease-in - animatsiya sekin boshlanadi

ease-out - animatsiya sekin tugaydi

ease-in-out - animatsiya sekin boshlanib va sekin tugaydi

cubic-bezier(x,y,z,k) - cubiz-bezier funksiyasi orqali, ko'rsatilgan egri chiziq animatsiya ega bo'ladi. x, y, z, k o'rniga qiymat ko'rsatiladi


animation-timing-function: ease-in-out;

Animation fill mode

Animatsiya boshlanishidan oldin va tugagandan so'ng, elementimiz o'zining oldingi xossalariga ega bo'ladi lekin quyidagi xossa orqali, bu holatni o'zgartirish mumkin

none - animatsiya tugagandan so'ng, elementning oldingi xossalariga ta'sir qilmaydi

forwards - element animatsiyaning oxirgi keyframe intervali tomonidan ko'rsatilgan xossalarga ega bo'ladi

backwards - element animatsiyaning birinchi keyframe intervali tomonidan ko'rsatilgan xossalarga ega bo'ladi

both - element animatsiyaning birinchi va oxirgi keyframe intervali tomonidan ko'rsatilgan xossalarga ega bo'ladi


animation-fill-mode: forwards;

Animatin play state

Animatsiya qanday holatda ekanini belgilab beradi. Uning 2 qiymat mavjud: running (animatsiya davom etayotgan holat) va paused (animatsiya to'xtab qo'yilgan holat)

Flexbox

Float yoki position xossalari yordamisiz veb sahifada moslashuvchan (flexible) layout yaratishda yordam beradi. Boshqacha qilib aytganda, veb sahifadagi elementlarni joylashtirish uchun xizmat qiladi. Flexbox 2 asosiy elementlardan tashkil topgan: flex container va flex items

veb sahifamizdagi flex container ota-onaga ega bo'lgan elementlar

asoslari quyidagilardan tashkil topgan: flex container, flex item, asosiy o'q (main axis), kesuvchi o'q (cross axis)

Justify content

Asosiy o'qi bo'ylab elementlarimizni joylashuvini belgilaydi va qo'shimcha bo'sh joyni taqsimlashga yordam beradi


flex-start - elementlarimiz flex container boshida joylashadi
flex-end - elementlarimiz flex container oxirida joylashadi
center - elementlarimiz flex container markazida joylashadi
space-between - elementlarimiz flex container bo'yicha teng taqsimlanadi. Birinchi elementimiz container boshida, oxirgi elementimiz esa container oxirida joylashadi
space-around - elementlarimiz flex container bo'yicha teng taqsimlanadi va har bir element atrofida teng masofa bo'ladi. Faqatgina birinchi, oxirgi element va container chegarali orasidagi masofa boshqa masofalardan farq qiladi

flex-end - elementlarimiz kesuvchi o'qning oxirida joylashadi

Order

Flex container ichida flex items oqimiga (flow) ko'ra joylashadi, ya'ni html kodda birinchi yozilgan flex item veb sahifada ham keladi, lekin order xossasi bu oqimni o'zgartirishi mumkin va flex items ketma-ketligini belgilab beradi.

Order xossasi son ko'rinishidagi qiymat qabul qiladi va qiymat qancha kichik bo'lsa, flex item boshqa "item"lardan oldinoq keladi

order: son qiymat;

ota-ona qutisida bo'sh qolgan joyni to'ldirish uchun ishlatiladi

Flex shrink

"flex item"ni kerak payti o'z o'lchovini qanday kichraytirishini belgilab beradi. U son ko'rinishidagi qiymat qabul qilib, joy qolmayothan payti element qanday suratda kichrayishini belgilab beradi. Default qiymati 1 hisoblanadi

CSS Grid

To'rga (setka) asoslangan layout bo'lib, float yoki position xossalari yordamisiz qatorlar va ustunlar yordamida veb sahifada layout yaratishda yordam beradi. Boshqacha qilib aytganda, veb sahifada elementlarni joylashtirish uchun xizmat qiladi. CSS Grid 2 asosiy elementlardan tashkil topgan: grid container va grid item.

Grid container

Display xossasi uchun grid yoki inline-grid qiymatiga ega bo'lgan elementga aytiladi.

Ikki asosiy xossasi mavjud: grid-template-columns va grid-template-rows


Grid template columns - container ichidagi ustunlar soni va uzunligini belgilashda ishlatiladi
Grid template rows - container ichidagi qatorlar balandligini belgilashda ishlatiladi

Grid item

"Grid item"ga tegishli xossalar yordamida elementni bir necha ustun va qatorlarni egallashini ko'rsatib o'tishimiz mumkin. Uning asosiy ikk xossasi mavjud: grid-column va grid-row.

Grid column - element nechta ustunni o'z ichiga olishini belgilab beradi.

grid-column: 1 / 3; - birinchi ustundan uchinchi ustungacha joyni egallashni angladatadi. [1;3).
grid-column: 1 / span 3; - nechta ustunni o'z ichiga olishini bildiradi. Bunda uchinchi ustunni hosobga oladi. [1;3]


Grid row - Element nechta qatorni o'z ichiga olishni belgilab beradi
grid-row: 1 / 3;
grid-row: 1 / span 2;
Yuqoridagilarni natijasi bir xil bo'ladi
